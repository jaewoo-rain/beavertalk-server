- `A-(으)ㄴ데, V-는데, N인데` :: 사과가 단데 수박은 싱거워요.
- `N마다` :: 월요일마다 운동해요.
- `V-는 게 어때요?` :: 점심에 김밥을 먹는 게 어때요?

- `A-아/어 보이다` :: 샌드위치가 맛없어 보여요.
- `N처럼/같이` :: 고양이처럼 조용히 걸어요.
- `A/V -(으)ㄴ 편이다` :: 소풍은 재미없는 편이에요.
- `A-게` :: 기차가 빠르게 지나가요.
- `A/V-(으)ㄹ지 모르겠다` :: 비가 올지 모르겠어요.
- `A/V-기는 하지만` :: 비가 오기는 하지만 산책할 거예요.
- `A/V/N (이)기 때문에` :: 폭우이기 때문에 학교에 가지 않았어요.
- `V-기(가) A` :: 지도를 읽기가 어려워요.
- `V-(으)ㄴ 적(이) 있다` :: 서울 미술관에 가 본 적이 있어요.
- `V-(으)ㄴ 적(이) 없다` :: 밤새 야영한 적이 없어요.
- `A/V-았/었을 때` :: 영화가 끝났을 때 관객이 박수쳤어요.

- `A-아/어지다` :: 신발이 편해졌어요.
- `V-게 되다` :: 한국 드라마를 보게 되었어요.
- `V-기 전에` :: 숙제를 하기 전에 손을 씻어요.
- `V-(으)ㄴ 후에` :: 밥을 먹은 후에 이를 닦아요.
